创建 config.py，包含：
- DATASET_ROOT = Path(r"D:\eeg\TDBRAIN-dataset")
- PARTICIPANTS_TSV = DATASET_ROOT / "participants.tsv"
- CONDITION = "EC"  # "EC" or "EO"

创建 data_loader.py，包含两个函数：

1. load_subjects(condition: str) -> pd.DataFrame
   - 读取 PARTICIPANTS_TSV，sep="\t"
   - 筛选：indication in ["MDD", "ADHD"] AND df[condition] == 1
   - 返回列：["participants_ID", "indication"]，reset_index(drop=True)
   - 必须用 indication 列（不是 formal_status）

2. load_raw(subject_id: str, condition: str) -> mne.io.Raw
   - 路径：DATASET_ROOT / subject_id / "ses-1" / "eeg" / f"{subject_id}_ses-1_task-rest{condition}_eeg.vhdr"
   - 调用 mne.io.read_raw_brainvision(str(vhdr), preload=False, verbose=False)
   - 若文件不存在，抛出 FileNotFoundError（含路径信息）

注意：
- 用 pathlib.Path 构建所有路径，传 str(path) 给 MNE（避免 Windows 路径问题）
- preload=False（520 个受试者全量 preload 会耗尽内存）
  </action>
