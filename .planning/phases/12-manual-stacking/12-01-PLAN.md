After the existing ensemble block (after line 136, before `return results`), add a stacking block.

The stacking block structure:

1. Initialize arrays:
   - `stacking_proba = np.zeros(len(y_enc))` for collecting test-set predictions
   - `full_oof = np.zeros((len(y_enc), len(FIXED_PIPES)))` for diagnostics
   - `stk_test_indices = []` for reordering

2. Outer CV loop — reuse the SAME `outer_cv` object:
   ```python
   for train_idx, test_idx in outer_cv.split(X, y_enc, groups):
   ```

3. Inner CV for OOF meta-features (inside each outer fold):
   - `X_train, y_train = X[train_idx], y_enc[train_idx]`
   - `X_test = X[test_idx]`
   - `g_train = groups[train_idx]`
   - `stacking_inner_cv = StratifiedGroupKFold(n_splits=3)`
   - `oof_meta = np.zeros((len(train_idx), len(FIXED_PIPES)))`
   - For each inner fold `(inner_train, inner_val)` from `stacking_inner_cv.split(X_train, y_train, g_train)`:
     - For each `(i, (name, pipe))` in `enumerate(FIXED_PIPES.items())`:
       - `cloned = clone(pipe)`
       - `cloned.fit(X_train[inner_train], y_train[inner_train])`
       - `oof_meta[inner_val, i] = cloned.predict_proba(X_train[inner_val])[:, mdd_idx]`
   - CRITICAL: Use `clone(pipe)` to avoid mutation. Use LOCAL indices (inner_train/inner_val are local to X_train).

4. Retrain base models on full train_outer for test-set meta-features:
   - `test_meta = np.zeros((len(test_idx), len(FIXED_PIPES)))`
   - For each `(i, (name, pipe))` in `enumerate(FIXED_PIPES.items())`:
     - `cloned = clone(pipe)`
     - `cloned.fit(X_train, y_train)`
     - `test_meta[:, i] = cloned.predict_proba(X_test)[:, mdd_idx]`

5. Train meta-learner on OOF, predict on test:
   - `meta_lr = LogisticRegression(C=1.0, solver="lbfgs", max_iter=1000, random_state=rs)`
   - `meta_lr.fit(oof_meta, y_train)` — NO SMOTE, NO scaler (per STK-03)
   - `stacking_proba[test_idx] = meta_lr.predict_proba(test_meta)[:, 1]`
   - Store test meta-features for diagnostics: `full_oof[test_idx] = test_meta`

6. Compute stacking metrics after all outer folds:
   - Reorder: `stk_order = np.argsort(np.concatenate(stk_test_indices))`
   - `stacking_proba = stacking_proba` (already indexed by test_idx, no reorder needed if using direct indexing)
   - `stk_auc = roc_auc_score(y_enc, stacking_proba)`
   - Youden optimal threshold (same pattern as lines 83-88):
     ```python
     fpr_s, tpr_s, thr_s = roc_curve(y_enc, stacking_proba, pos_label=mdd_idx)
     stk_opt_idx = int(np.argmax(tpr_s - fpr_s))
     stk_opt_thresh = float(thr_s[stk_opt_idx])
     stk_pred = np.where(stacking_proba >= stk_opt_thresh, mdd_idx, 1 - mdd_idx)
     ```
   - Confusion matrix: `tn_s, fp_s, fn_s, tp_s = confusion_matrix(y_enc, stk_pred).ravel()`
   - SEN/SPE/BA at optimal threshold

7. Compute diagnostics:
   - Fit diagnostic meta-learner on full_oof: `meta_lr_diag = LogisticRegression(C=1.0, solver="lbfgs", max_iter=1000, random_state=rs)`
   - `meta_lr_diag.fit(full_oof, y_enc)`
   - Extract coefficients: `dict(zip(FIXED_PIPES.keys(), meta_lr_diag.coef_[0].round(4).tolist()))`
   - OOF single-model AUC: `{name: round(float(roc_auc_score(y_enc, full_oof[:, i])), 4) for i, name in enumerate(FIXED_PIPES.keys())}`

8. Build results["stacking"] dict matching existing structure:
   ```python
   results["stacking"] = {
       "auc": float(stk_auc),
       "accuracy": float(accuracy_score(y_enc, stk_pred)),  # standard accuracy, same as other models
       "sensitivity": float(stk_sen),
       "specificity": float(stk_spe),
       "optimal_threshold": stk_opt_thresh,
       "meta_learner_coefs": coefs_dict,
       "oof_base_auc": oof_aucs_dict,
       "permutation_pvalue": None,
       "n_splits": int(n_splits),
   }
   ```

9. Add comparison field and console warning:
   ```python
   ens_auc = results["ensemble"]["auc"]
   ens_ba = (results["ensemble"]["sensitivity"] + results["ensemble"]["specificity"]) / 2
   stk_ba = (stk_sen + stk_spe) / 2
   delta_auc = round(float(stk_auc) - ens_auc, 4)
   delta_ba = round(stk_ba - ens_ba, 4)

   results["stacking"]["stacking_vs_ensemble"] = {
       "delta_auc": delta_auc,
       "delta_ba": delta_ba,
       "stacking_wins": float(stk_auc) >= ens_auc,
   }

   if float(stk_auc) < ens_auc:
       print(f"WARNING: Stacking AUC ({stk_auc:.3f}) < soft-vote AUC "
             f"({ens_auc:.3f}). Delta = {delta_auc:.4f}")
   ```
