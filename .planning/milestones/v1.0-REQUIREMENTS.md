# Requirements: TDBRAIN MDD vs ADHD 鉴别诊断

**Defined:** 2026-02-23
**Core Value:** 用频谱特征在 TDBRAIN 上验证 MDD vs ADHD 分类可行性，AUC > 0.70 且置换检验显著

## v1 Requirements

### 数据加载

- [x] **DATA-01**: 从 participants.tsv 筛出 MDD（320例）和 ADHD（200例）受试者列表
- [x] **DATA-02**: 加载对应的静息态 EEG 文件（.vhdr 格式，BIDS 结构）
- [x] **DATA-03**: 支持 EC（闭眼）和 EO（睁眼）两种条件，可配置选择

### 预处理

- [ ] **PREP-01**: 基础预处理：带通滤波（1-40 Hz）、重参考（平均参考）
- [ ] **PREP-02**: 分段（epoch）：静息态切成固定长度片段（如 2s）
- [ ] **PREP-03**: 简单伪迹剔除：振幅阈值法（±100 µV）

### 特征提取

- [ ] **FEAT-01**: 各频段绝对功率（delta/theta/alpha/beta/gamma）
- [ ] **FEAT-02**: Theta/Beta ratio（额叶通道，ADHD 核心指标）
- [ ] **FEAT-03**: Frontal Alpha Asymmetry（F3/F4，MDD 核心指标）
- [ ] **FEAT-04**: 相对功率（各频段功率 / 总功率）

### 分类与验证

- [ ] **CLF-01**: StratifiedGroupKFold 交叉验证（按 subject 分组，防数据泄露）
- [ ] **CLF-02**: 分类器：SVM（RBF kernel）+ 标准化
- [ ] **CLF-03**: 报告指标：AUC、准确率、敏感性、特异性
- [ ] **CLF-04**: 1000 次置换检验，报告 p 值

### 输出

- [ ] **OUT-01**: 控制台打印结果摘要（AUC、p 值、置换检验结论）
- [ ] **OUT-02**: 保存结果到 JSON 文件（可复现）

## v2 Requirements

### ERP 特征

- **ERP-01**: P300 潜伏期和幅值（需要 oddball 任务数据）
- **ERP-02**: N200 成分

### 功能连接

- **CONN-01**: 电极间相干性（coherence）
- **CONN-02**: 相位同步指数

### 模型增强

- **MODEL-01**: 深度学习分类器（CNN/LSTM）
- **MODEL-02**: 多模态特征融合

## Out of Scope

| Feature | Reason |
|---------|--------|
| 实时推断 | 纯研究验证，不需要在线处理 |
| 临床部署 | 超出可行性验证范围 |
| 其他诊断组（OCD/TINNITUS等） | 聚焦 MDD vs ADHD 二分类 |
| 药物响应预测 | 超出当前目标 |
